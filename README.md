# FoodClaw

Restaurant and food service management for [ERPClaw](https://github.com/avansaber/erpclaw). Menus, recipe costing, F&B inventory, staff scheduling, catering, HACCP food safety, and franchise management. 71 actions across 7 domains.

## Features

- **Menu Management** -- menus, menu items, modifier groups, dietary flags (vegetarian, vegan, gluten-free)
- **Recipe Costing** -- recipes with ingredients, cost calculation, scaling, food cost analysis
- **F&B Inventory** -- ingredients with par levels, stock counts, waste logging, purchase orders
- **Staff Scheduling** -- employee roles, shift management, clock in/out, tip distribution
- **Catering** -- events, catering items, dietary requirements, cost estimates, confirmation workflow
- **Food Safety** -- HACCP logs, temperature readings, health inspections, violation alerts, compliance reports
- **Reports** -- food cost, labor, waste, menu performance, franchise comparison, daily sales

## Installation

Requires [ERPClaw](https://github.com/avansaber/erpclaw) core.

```
install-module foodclaw
```

## Quick Start

```
"I'm opening a restaurant -- set me up"
```

The bot will install FoodClaw and guide you through creating menus, building recipes, tracking inventory, and scheduling staff.
